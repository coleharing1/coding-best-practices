# Macro Roadmap

- Treat `plans/MACRO-ROADMAP.md` as the whole-project roadmap.
- Read it before large planning or review work.
- Every non-trivial `Plan-XXX-*.md` should include a `Macro Roadmap Alignment` section with roadmap IDs.
- If synthesis changes sequencing, durable assumptions, likely-next work, or roadmap status, update `plans/MACRO-ROADMAP.md`.
- Keep build-sized execution detail in `plans/active/Plan-XXX-Final.md`, not in the macro roadmap.
